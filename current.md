   * Publier des fragments d’expérience dans l’aperçu : pour prévisualiser les expériences finales que vos visiteurs peuvent voir, vous pouvez publier des fragments d’expérience autonomes sur le service d’aperçu as a Cloud Service Experience Manager.

* **Experience Manager Forms as a Cloud Service**

   _Nouvelles fonctionnalités_
